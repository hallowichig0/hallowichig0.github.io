== Description ==

Remove Customizer Menu allow you to hide the navigation menu from the Customizer.

== Installation ==

WordPress (Also works on multisite enabled instance):

1. Copy the `remove-customizer-menu` folder into your `wp-content/plugins` folder
2. Activate the remove-customizer-menu plugin via the plugins admin page

== Frequently Asked Questions ==

= What if I deactivate this plugin? =

The Remove Customizer Menu will be changed back navigation menu from the Customizer, there's no database writes with this plugin.
